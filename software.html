Software developed

Materials science

  • PyMCSP: a software for crystal structure prediction and diffraction study.
  • Materials Modelling tools: Scripts and utilities for computational materials science modelling.
  • Crystal Graph Neural Networks: Code to generate graphs from a unit cell of a crystal material and train a graph neural network to predict properties of materials.
  • Tight Binding models: Code to solve basic tight binding models.
  • Excitons models: Code for modelling exciton systems in 2d-materials.
